# 이미지 프롬프트 가이드

소설 관련 비주얼 자료를 생성하기 위한 프롬프트 템플릿.
canvas-design, html-to-image 스킬과 연계하여 사용한다.

---

## 1. 캐릭터 비주얼

```
[캐릭터명] 초상화
- 외모: {외모 묘사}
- 복장: {의복 설명}
- 표정: {감정 상태}
- 배경: {장소 또는 상징적 요소}
- 스타일: {사실적/만화/수채화/등}
- 시점: {정면/측면/3/4}
```

## 2. 장소/배경

```
[장소명]
- 구조: {건물/지형 설명}
- 조명: {시간대, 광원}
- 분위기: {으스스한/따뜻한/웅장한}
- 날씨: {맑음/비/안개}
- 디테일: {특징적 요소}
- 스타일: {컨셉 아트/사실적/인상파}
```

## 3. 핵심 씬

```
[챕터X 장면명]
- 등장인물: {위치/행동}
- 장소: {배경 설명}
- 시간: {시간대/계절}
- 분위기: {긴장/평화/혼돈}
- 핵심 요소: {씬에서 가장 중요한 시각 요소}
- 구도: {광각/클로즈업/조감도}
```

## 4. 표지 디자인

```
장르: {장르}
톤: {어두운/밝은/신비로운}
핵심 이미지: {표지에 들어갈 주요 요소}
색상 팔레트: {주색/보조색}
타이틀 위치: {상단/중앙/하단}
스타일: {출판 표지/웹소설 표지}
```
