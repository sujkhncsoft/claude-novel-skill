# 캐릭터 시스템 가이드

캐릭터의 생성, 관리, 추적을 위한 체계적 시스템.
일관성 있는 캐릭터를 유지하고, 성장 아크를 설계하며, 관계 변화를 추적한다.

---

## 1. 캐릭터 프로필 구조

### 1-1. 필수 요소

| 계층 | 항목 | 설명 |
|------|------|------|
| 정체성 | 이름, 별칭, 역할 | 작품 내 기능적 위치 |
| 외면 | 외모, 체형, 특징 | 시각적 식별자 |
| 내면 | 성격, 결함, 두려움, 욕망, 가치관 | 행동 동기의 근원 |
| 배경 | 출생, 교육, 직업, 핵심 사건 | 현재 성격의 원인 |
| 아크 | 시작 상태, 전환점, 목표 상태 | 성장/변화 궤적 |

### 1-2. 욕망-두려움 매트릭스

모든 주요 캐릭터는 다음 4가지를 명확히 정의해야 한다:

```
외적 욕망 (Want): 캐릭터가 의식적으로 추구하는 것
내적 욕망 (Need): 캐릭터가 진정으로 필요하지만 인식하지 못하는 것
외적 두려움 (External Fear): 구체적 위협
내적 두려움 (Internal Fear): 심리적 한계
```

Want와 Need의 갈등이 캐릭터 아크의 핵심 동력이 된다.

### 1-3. 아크타입 활용

| 아크타입 | 핵심 특성 | 변형 가능성 |
|----------|----------|------------|
| 영웅 | 사명감, 자기희생 | 안티히어로, 마지못한 영웅 |
| 멘토 | 지혜, 경험 | 타락한 멘토, 부재 멘토 |
| 트릭스터 | 변칙, 혼돈 | 도움주는 사기꾼, 진지한 광대 |
| 수호자 | 충성, 보호 | 맹목적 충성, 배신자 |
| 그림자 | 대립, 반영 | 동정적 악역, 거울 캐릭터 |

아크타입은 출발점이지, 도착점이 아니다. 의식적으로 전복하거나 혼합하라.

## 2. 캐릭터 음성 (Voice)

### 2-1. 대사 차별화 요소

| 요소 | 적용 예시 |
|------|----------|
| 어휘 수준 | 학자 vs 거리 출신 |
| 문장 길이 | 과묵한 인물(짧게) vs 수다스러운 인물(길게) |
| 말버릇/감탄사 | 캐릭터 고유 표현 |
| 화법 패턴 | 질문형, 명령형, 비유형 |
| 사투리/방언 | 지역/계층 반영 |
| 금기어 | 이 캐릭터가 절대 쓰지 않는 말 |

### 2-2. 대사 일관성 검증

씬 작성 후 다음을 확인한다:
- 이 대사를 다른 캐릭터 이름으로 바꿔도 어색하지 않다면, 차별화 실패
- 캐릭터의 교육/배경에 어울리지 않는 어휘를 사용했는지 확인
- 감정 상태에 따른 말투 변화가 반영되었는지 확인

## 3. 관계 추적

### 3-1. 관계 매트릭스

캐릭터 쌍마다 다음을 기록한다:

```
A -> B 감정: 존경/적대/애정/무관심/경계/...
B -> A 감정: (비대칭 가능)
관계 변화 이력: [{챕터, 사건, 이전감정, 이후감정}]
핵심 갈등: 두 캐릭터 간 근본적 충돌 지점
```

### 3-2. 관계 변화 트리거

관계는 사건을 통해서만 변한다. 갑작스러운 변화는 다음 중 하나를 요구한다:
- 충격적 사실 발견
- 극한 상황에서의 행동 목격
- 직접적 배신 또는 희생
- 점진적 축적 후 임계점 돌파

---

## 4. 캐릭터 아크 설계

### 4-1. 아크 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 긍정적 변화 | 결함 극복, 성장 | 비겁함 -> 용기 |
| 부정적 변화 | 타락, 추락 | 이상주의 -> 냉소주의 |
| 평탄 아크 | 변하지 않고 세상을 바꿈 | 신념 유지로 주변 감화 |
| 순환 아크 | 변화했다가 원점 회귀 | 비극적 반복 |

### 4-2. 아크 진행 체크포인트

```
[기] 결함 상태 제시 (1막)
[기] 변화의 필요성 암시 (촉매 사건)
[승] 변화 시도와 실패 반복 (2막 전반)
[전] 핵심 전환점 -- 진실 직면 (2막 후반)
[결] 변화 완성 또는 거부 (3막)
```

각 체크포인트를 novel-brain.json의 arc.turningPoints에 기록한다.

## 5. 등장 빈도 관리

### 5-1. 캐릭터 등장 규칙

- 주인공: 모든 챕터 또는 교대 시점 구조
- 주요 조연: 3챕터 이상 연속 미등장 시 경고
- 적대자: 1막 후반, 2막 중반, 3막 시작에 반드시 등장
- 배경 인물: 등장 시마다 최소 1개 식별 가능 특성 부여

### 5-2. 소외 캐릭터 감지

주간 점검 시 다음을 확인한다:
- 5챕터 이상 언급되지 않은 주요 캐릭터
- 등장했지만 플롯에 기여하지 않는 캐릭터
- 서브플롯이 정체된 캐릭터

---

## 6. AI 협업 시 캐릭터 관리

### 6-1. 캐릭터 목소리 생성

`/novel voice [이름]` 명령 시:
1. 해당 캐릭터 프로필의 voice 섹션을 로드
2. 최근 3씬의 대사 패턴을 분석
3. 현재 감정 상태를 반영
4. 상황에 맞는 대사 후보 3개 생성

### 6-2. 캐릭터 일관성 검증

`/novel check` 실행 시 검증 항목:
- 프로필의 성격 특성과 행동 일치 여부
- 음성 패턴 일탈 감지
- 아직 모르는 정보를 언급하는 지식 누수 감지
- 관계 변화 없이 태도가 바뀐 경우 감지
