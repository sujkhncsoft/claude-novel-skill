# 페이싱/텐션 가이드

소설의 리듬을 설계하고, 긴장의 곡선을 의식적으로 조절한다.

---

## 1. 텐션 곡선

### 1-1. 기본 원칙

텐션은 독자가 "다음에 무슨 일이 일어날까"를 궁금해하는 정도다.
단조로운 텐션은 지루함을, 끊임없는 고텐션은 피로를 유발한다.

### 1-2. 이상적인 텐션 패턴

```
높이 ^
     |      *         *    ****
     |    *   *     *  * **    *
     |  *       * *      *      *
     | *         *                *
     +----------------------------> 시간
     1막      2막전반   2막후반   3막
```

핵심: 상승과 이완의 교대. 각 이완 구간은 이전 정점보다 높은 바닥을 가진다.

## 2. 씬 레벨 페이싱

### 2-1. 씬 텐션 등급

| 등급 | 설명 | 예시 |
|------|------|------|
| 5 (극한) | 생사의 기로, 최종 대결 | 클라이맥스 전투, 폭로 |
| 4 (높음) | 중대한 위협, 강한 갈등 | 추격, 대립, 배신 |
| 3 (보통) | 진행 중인 갈등, 목표 추구 | 탐색, 조사, 계획 |
| 2 (낮음) | 캐릭터 교류, 관계 발전 | 대화, 일상, 휴식 |
| 1 (이완) | 호흡 조절, 감정 정리 | 풍경, 회상, 성찰 |

### 2-2. 씬 배열 규칙

- 등급 5 씬 다음에는 반드시 등급 1-2 씬 배치 (호흡)
- 3개 이상 같은 등급이 연속되면 단조로움 경고
- 챕터 내에서 텐션은 상승 추세를 유지한다
- 챕터 마지막 씬은 현재 등급보다 1단계 이상 높여 끝낸다

## 3. 챕터 레벨 페이싱

### 3-1. 챕터 엔딩 훅

모든 챕터는 다음 중 하나로 끝난다:

| 훅 유형 | 설명 |
|---------|------|
| 클리프행어 | 위기 상황에서 끊기 |
| 반전 공개 | 새로운 정보/사실 폭로 |
| 결정의 순간 | 캐릭터가 선택을 앞둔 상태 |
| 미스터리 질문 | 새로운 의문 제기 |
| 감정 정점 | 강한 감정이 해소되지 않은 상태 |

### 3-2. 챕터 오프닝

이전 챕터의 훅을 즉시 해소하지 않는다:
- 다른 시점/장소로 전환하여 궁금증 유지
- 해소하더라도 새로운 문제를 즉시 제시

## 4. 전체 구조 페이싱

### 4-1. 에너지 분배

| 구간 | 텐션 범위 | 목적 |
|------|----------|------|
| 1막 (1-25%) | 1-3 | 세계/인물 몰입 구축 |
| 촉매 (10-15%) | 3-4 | 사건 시동 |
| 2막 전반 (25-50%) | 2-4 | 상승 + 이완 교대 |
| 중간점 (50%) | 4-5 | 반전/에스컬레이션 |
| 2막 후반 (50-75%) | 3-5 | 압축, 이완 축소 |
| 최저점 (75%) | 5 (역설적 정지) | 모든 것을 잃음 |
| 3막 (75-100%) | 4-5 | 지속적 상승 -> 해소 |

### 4-2. 이완 구간의 역할

이완 씬은 "쉬는 시간"이 아니다. 다음 기능을 수행한다:
- 캐릭터 깊이 확보 (대화, 관계)
- 다음 갈등을 위한 복선 심기
- 테마 탐구
- 세계관 디테일 자연스럽게 전달

## 5. 웹소설 특화 페이싱

### 5-1. 회차별 구조

```
훅 오프닝 (첫 5줄) -> 전개 -> 미니 클라이맥스 -> 훅 엔딩
```

- 첫 5줄 안에 흥미를 잡아야 한다
- 회차 중간에 최소 1개의 텐션 피크
- 마지막 문장은 반드시 다음 회차로 견인

### 5-2. 에피소드 아크

10-20회차 단위의 미니 아크:
- 독립적 갈등 + 해소
- 전체 아크에 기여하는 진전
- 에피소드 종료 시 더 큰 질문 제기

---

## 6. 텐션 도구

| 도구 | 설명 |
|------|------|
| 시한장치 | 시간 제한 부여 (마감, 카운트다운) |
| 정보 비대칭 | 독자가 알고 캐릭터가 모르는 것 |
| 딜레마 | 어느 쪽도 좋지 않은 선택 |
| 미스터리 | 답을 모르는 질문 |
| 감정 위협 | 관계/가치의 상실 위험 |
| 물리적 위험 | 신체적 위험, 생존 위기 |
